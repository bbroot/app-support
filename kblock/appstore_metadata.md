## Description (描述)

**Stop fighting with your input source.**

KBLock is a macOS menu-bar utility that pins your keyboard input source, so it never jumps between Chinese and English at the wrong moment.

Whether you're coding in VS Code (need English), chatting in WeChat (need Chinese), or running a terminal command — KBLock ensures the right input source is active, automatically.

### Key Features

• **Global Lock** — Pin to one input source system-wide. Your keyboard behaves the same everywhere.

• **Per-App Rules** — Auto-lock to a specific source when a particular app is frontmost. Terminal, VS Code → English; WeChat → Chinese. Add rules manually or let KBLock learn from your usage.

• **Smart Suggestions** — The more you use KBLock, the smarter it gets. It learns which apps use which sources and suggests rules based on your actual behavior.

• **Usage Statistics** — Visual breakdown of how much you use each input source, per app, over time. See your typing habits at a glance.

• **iCloud Sync** (Beta) — Your rules sync across all your Macs automatically. Set it up once, use it everywhere.

• **Lightweight & Native** — Pure SwiftUI, runs as a menu-bar app. Minimal CPU and memory footprint. Built for macOS 14 Sonoma and later.

• **Multi-Language** — Localized in English, Chinese, Japanese, Korean, French, German, Spanish, Portuguese, and Russian.

### Why KBLock?

macOS input source switching is smart — until it's not. KBLock gives you back control without adding complexity. No configuration overload. Just set your target source and forget it.

### Requirements
- macOS 14 Sonoma or later
- Apple Silicon or Intel (universal binary)
