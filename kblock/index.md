# KBLock — Keep your keyboard locked.

Fix your macOS input source. No more typing Chinese in terminals.

## Description

**KBLock** is a macOS menu-bar utility that pins your keyboard input source, so it never jumps between Chinese and English at the wrong moment.

Whether you're coding in VS Code (need English), chatting in WeChat (need Chinese), or running a terminal command, KBLock ensures the right input source is active — automatically.

### Key Features

- **Global Lock**: Pin to one input source system-wide. Your keyboard behaves the same everywhere.
- **Per-App Rules**: Auto-lock to a specific source when a particular app is frontmost. Terminal, VS Code → English; WeChat → Chinese.
- **Smart Suggestions**: Learns which apps use which sources and suggests rules based on your usage patterns.
- **Usage Statistics**: See how much you use each input source, per app, and when.
- **iCloud Sync** (beta): Your rules sync across all your Macs automatically.
- **Lightweight & Native**: Built with SwiftUI, runs as a menu-bar app. Minimal CPU and memory usage.

### Use Cases

- Developers who accidentally type Chinese in the terminal or IDE
- Bilingual users switching between Chinese and English frequently
- Writers who need consistent input source behavior per app
- Anyone tired of the macOS input source randomly switching

### Requirements

- macOS 14 Sonoma or later
- Apple Silicon or Intel (universal binary)
