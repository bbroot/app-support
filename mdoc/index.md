# MDoc - Markdown to Word

**MDoc** is a simple and fast Markdown to Word converter for iOS.

## Description

MDoc helps you convert Markdown documents to professional Word (.docx) format with support for:
- Multiple built-in templates (Standard, Academic, SCI Paper, Technical, Official Document)
- Custom templates with adjustable fonts, styles, margins, and more
- Real-time Markdown preview with full-screen mode
- Export to Word with proper formatting

MDoc does not collect any personal data. All documents are stored locally on your device.
